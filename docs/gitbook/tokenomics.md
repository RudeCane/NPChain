# Tokenomics

## Certs — The Native Currency

NPChain's native token is the **Cert** (symbol: CERT). One Cert has 6 decimal places (1,000,000 base units).

## Emission Schedule

| Parameter | Value |
|-----------|-------|
| Annual emission | 100 billion Certs |
| Block reward (testnet, 15s) | ~47,564 Certs |
| Block reward (mainnet, 60s) | ~190,258 Certs |
| Halving | **None** — constant emission |
| Maximum supply | **No hard cap** — controlled by fee burn |

### Why No Halving?

Bitcoin's halving creates supply shocks every 4 years. NPChain uses constant emission with deflationary pressure from fee burn instead. Miners always know what they'll earn. The network's security budget never drops.

## EIP-1559 Fee Burn

Every transaction pays a base fee that is burned (destroyed). As usage grows, more Certs are burned. When burn rate exceeds emission, Certs become deflationary.

## Testnet to Mainnet Migration

Testnet Certs convert at **1000:1 ratio**. Minimum ~47,564 testnet Certs to be eligible. KYC required. Merkle proof in mainnet genesis block.

## Distribution

No pre-mine. No ICO. No VC allocation. 100% earned through mining.

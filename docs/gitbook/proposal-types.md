# Proposal Types

## Parameter Change

Modify consensus parameters like block time, difficulty adjustment window, or reward schedule. Requires specifying the parameter key and new value.

## Crypto Upgrade

Add or deprecate cryptographic algorithms. Used when NIST standardizes new post-quantum algorithms or when existing ones need updating. Requires independent audits before voting.

## Fee Change

Adjust the EIP-1559 fee structure — base fee multiplier, burn rate, or tip mechanics.

## Emission Change

Modify the annual emission cap or block reward calculation. The most impactful proposal type — affects all token holders.

## General

Any other governance decision that doesn't fit the above categories. Used for ecosystem-level decisions, partnerships, or protocol philosophy changes.
